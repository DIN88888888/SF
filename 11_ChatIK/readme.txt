3. клиент-сервер
   не нашёл способа нормально отображать ткст сообщений и ввод пользователя в консоль, не применяя потоки
   решил сделать некоторое подобие интерфейса с помощью отрисовки в буфер консоли.

после запуска программы выберите нужное действие клаиатурой.
сверху будут подсказки о возможных действиях.

4.

TMainController - основной объект системы
TScreen - интерфейс отрисовки в буфер консоли
TServer - служба сервера
TServer_Connection - подключение клиента на стороне сервера
TServerDataClass - накопитель данных сервера
IClient - служба клиента
TClientDataClass - накопитель данных клиента
TEventQue - обменник сообщениями между потоками
   IEvent - основа
   TEvent_Key - событие клавиатуры
   TEvent_DebugText - вывод текста отладки
   TEvent_net_send_ServerName - отправка имени сервера клиентам
   TEvent_net_send_LoginPass - данные авторизации от клиента на сервер
   TEvent_net_send_AuthStatus - результат попытки авторизации с сервера на клиент
   TEvent_net_send_HistMesage - отправка строки сообщения с сервера на клиент.
   TEvent_net_send_Mesage - отправка сообщения с клиента    
   TEvent_net_send_Disconnect - сообщение об отключении клиента
TUI_Main - главный обработчик интерфейса - управляет вкладками
вкладки:
   IUI_Tab - основа
   TUI_Tab_Main - первая страница программы
   TUI_Tab_ServerOptions - настройки подключения сервера
   TUI_Tab_ClientConnection - соединение с сервером
   TUI_Tab_ClientAuth - авторизация
   TUI_Tab_Chat - чат.
интерфейс элементов интерфейса... :
   IUI_Item - основа
   IUI_Item_base - что-то имеющее размеры
   IUI_Item_Selectable - выделяемое
   IUI_Item_Action - можно активировать для выполнения действия
   IUI_Item_Btn - интерфейс кнопки
   IUI_Item_TextEdit - ввод текста
реализация элементов интерфейса:
   TUI_Item_Text - рисует текст в заданных рамках
   TUI_Item_Rect - окрашивает прямоугольник
   TUI_Item_Btn - кнопка
   TUI_Item_ServerStatus - кнопка-сервер, отображает его статус
   TUI_Item_TextEdit - ввод текста
   TUI_Item_ClientStatus - кнопка - статус клиента
   TUI_Item_ChatEdit - ввод текста чата
   TUI_Item_Chat final - отображение сообщений чата
